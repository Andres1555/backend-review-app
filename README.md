## Ejecución

Clona el proyecto

```bash
  git clone https://github.com/NetKBs/backend-review-app.git
```

Accede a la ruta del proyecto

```bash
  cd backend-review-app
```

Crea las variables de entorno en tu .env

```bash
  mkdir config 
  touch .env
```

Instala las dependencias

```bash
  go mod download
```

Inicia el servidor
```bash
  go run main.go
```
